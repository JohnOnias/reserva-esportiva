para rodar  o projeto na sua maquina faça:

passo 1: crie um arquivo ".env"
passo 2: copie o conteudo do arquivo ".env.example" para dentro do ".env"
passo 3: instale o composer no projeto, rode o comando  "composer install" dentro da pasto do projeto  
passo 4: crie um banco de dados com o mesmo nome e configurações que esta no ".env" e use o comando "php artisan migrate"
passo 5: caso necessario rode o comando "php artisan key:generate" 

caso não rode mesmo assim,  Faça o L e se vira :D
